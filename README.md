# Dream Foundry · 梦境熔炉

Dream Foundry takes four subconscious inputs (fears, recent memories, present desires, dominant emotional weather) and runs them through eight recurring archetypes and a twelve-entry symbolic lexicon to forge oneiric narratives. Bilingual (EN / 中文), dark/light, custom psychological brand (Cormorant Garamond italic display + Lora body + JetBrains Mono on twilight `#0c0820` with dream-violet `#8a6dff` plus twilight-pink, mist-cyan, oneiric-rose, sigil-gold, abyss-indigo accent codes). Hero centerpiece: animated multi-ring SVG dream-orb with 夢 character at the still point. Five modules: D1 Eight Archetypes (chase, falling, flying, lost-place, drowning, found-room, dead-loved-one, naked-public — each as starting reading not a key), D2 The Foundry (4 inputs — fear/memory/desire sliders + emotional-weather dropdown — plus three-archetype selector → forges short oneiric passage with palette-keyed light and intensity-keyed closing), D3 Symbolic Lexicon (12 symbols × common reading × cost-of-misread line), D4 Oneiric Engine (5 multi-paragraph canned probes — recurring dream, lucid asymmetry, no-dreams-at-all, stuck nightmare, shared dream — plus free-text heuristic fallback that detects 9 imagery types plus recurrence and fear vocabulary), D5 Guardrails (explicit refusal of prophecies, deterministic dream-key tables, Freudian determinism, clinical claims, infinite-AI-generation slop).

## Stack

Plain HTML, CSS, vanilla JavaScript. No build step. Bilingual via `[data-lang]` + CSS attribute selectors. Light/dark via `[data-theme]`. Cormorant Garamond italic display, Lora body, JetBrains Mono mono, Noto Serif SC for Chinese. All visualisations inline SVG. JS verified with `node --check`.
